# api-speech
A Eleven labs (text to speech) wrapper, recibe text in the body and consume eleven labs api to generate audio then return it to the client.

## Installation
- npm i
- You need this env vars.
    ```bash
    API_LOG_LEVEL='short'
    API_NAME='speech'
    NAME_SPACE='xylon'
    PORT=3000
    ELEVEN_API_KEY='XxxElevenLabskeyXxx'

## Usage
- npm start
